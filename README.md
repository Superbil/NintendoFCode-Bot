用來記錄大家的 Nintendo Switch Friend Code 唷~  

## 用法

-   新增自己的好友代碼  

    /addfc SW-1234-5678-9102 

-   顯示自己的好友代碼  

    /myfc

-   搜尋別人的好友代碼  

    /findfc @tdcccc 

-   使用說明  

    /help
